## Controls 🕹️
- ***Left Arrow Key*** - Left Movement
- ***Right Arrow Key*** - Right Movement
- ***Enter*** - Start/Interact with Environment (e.g. door or PC)
- ***P*** - Pause Game
- ***Q*** - Quit Level
- ***R*** - Reset Level
